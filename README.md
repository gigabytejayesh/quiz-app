## Setup
### Prerequisites

- Node.js (version ^16.13.1)

### Installation
1. Install dependencies:
    ```bash
    npm install

2. Start the server:
    ```bash
    npm start

The server will be running on http://localhost:3000.
To access Swagger Documention - http://localhost:3000/api-docs
